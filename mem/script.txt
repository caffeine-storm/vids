Here's how you can think about "memory"

- Imagine a number line

- Let's ignore all the negative numbers
- Now, let's give each whole number a box
- But it'll be easier to draw like this with zero at the top

- Each of these boxes can, itself, has one number in it

Now you can do two things
	1. look at what number is in a box
	2. put a number in a box
		- but, because there can only be one number, we have to replace whatever is there
